getAlbumTracks

final SpotifyRestActionPaging<SimpleTrack, PagingObject<SimpleTrack>> getAlbumTracks(String album, Integer limit, Integer offset, Market market)
Content copied to clipboard

Get Spotify catalog information about an album’s tracks. Optional parameters can be used to limit the number of tracks returned.

Api Reference

Return

PagingObject of SimpleTrack objects

Parameters

album

The com.adamratzman.spotify id or uri for the album.

limit

The number of objects to return. Default: 50 (or api limit). Minimum: 1. Maximum: 50.

offset

The index of the first item to return. Default: 0. Use with limit to get the next set of items

market

Provide this parameter if you want to apply Track Relinking

Throws

com.adamratzman.spotify.SpotifyException.BadRequestException

if the album is not found, or positioning of limit or offset is illegal.
