getPlaylistsForCategory

final PagingObject<SimplePlaylist> getPlaylistsForCategory(String categoryId, Integer limit, Integer offset, Market market)
Content copied to clipboard

Get a list of Spotify playlists tagged with a particular category.

Api Reference

Return

PagingObject of top playlists tagged with categoryId

Parameters

market

Provide this parameter if you want the list of returned items to be relevant to a particular country. If omitted, the returned items will be relevant to all countries.

limit

The number of objects to return. Default: 50 (or api limit). Minimum: 1. Maximum: 50.

offset

The index of the first item to return. Default: 0. Use with limit to get the next set of items

Throws

com.adamratzman.spotify.SpotifyException.BadRequestException

if categoryId is not found or filters are illegal
